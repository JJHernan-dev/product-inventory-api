## 📦 PRODUCT INVENTORY

REST API build with Spring Boot for managing product inventory, including CRUD operations, stocks updates, filtering, and categories.

## Features
- Create, read, update and delete products
- Manage stocks
- Filter by price and name
- Category management

## Tech Stack
- Java
- Spring Boot
- Maven
